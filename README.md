🛡️ LinkShield: Advanced Web Crawler & Security ScannerLinkShield is a high-performance C++ web crawler designed to identify broken links, detect malicious URLs using blacklist hashing, and provide "smart-fix" suggestions for dead pages using string distance algorithms.🚀 FeaturesBFS-Based Crawling: Systematically explores web structures using Breadth-First Search.Malicious Domain Detection: High-speed lookup against security blacklists.Smart Suggestions: Uses Levenshtein Distance to suggest correct URLs for 404 errors.Real-time Dashboard: Local web interface powered by the Crow framework.High Performance: Optimized with hash-based data structures for $O(1)$ lookups.📊 Data Structures & Complexity AnalysisThis project was built with a focus on algorithmic efficiency. Below is the breakdown of the primary data structures used:ComponentData StructureTime ComplexityWhy?Crawl Frontierstd::queue$O(1)$Standard FIFO behavior for Breadth-First Search.Visited Registrystd::unordered_set$O(1)$ (avg)Hashing ensures we never scan the same page twice, regardless of site size.Web Graphstd::unordered_map$O(1)$ (avg)Stores the site map as an Adjacency List using hash-keys.Priority Scanstd::priority_queue$O(\log n)$Prioritizes "high-value" pages (like /login or /cart) during the crawl.Typo CorrectionLevenshtein Automata$O(m \cdot n)$Calculates the minimum "edits" needed to suggest a fix for a broken link.🛠️ Tech StackLanguage: C++17Web Framework: Crow (Fast, header-only web microframework)Networking: libcurl for robust HTTP requests.HTML Parsing: Gumbo Parser (Google's pure-C HTML5 parser).Async I/O: Asio for handling concurrent dashboard requests.📥 Installation (macOS)Install DependenciesBashbrew install crow asio gumbo-parser curl
